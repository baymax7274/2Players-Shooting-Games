## v1.2.0 (2026-05-10)

### 新增 — 动态环境系统
- **拾取物系统**：地图随机刷新血包(+25HP)、伤害翻倍(1.5x/8s)、加速(1.5x/6s)，每张地图10个刷新点
- **可破坏墙壁**：火箭弹/手雷爆炸可摧毁可破坏墙，霰弹枪近距离单发摧毁，摧毁时生成碎片粒子
- **半高掩体**：阻挡移动但子弹可越过，提供战术掩护位
- **移动不可破坏墙**（训练靶场）：沿预设路径往复移动，碰到玩家可推动，表面显示移动方向箭头
- **地图专属机制**：
  - 训练靶场 — 移动掩体 ×2
  - 城市废墟 — 随机塌方（每25-35s自动坍塌一块边缘墙体）
  - 地下堡垒 — 毒气扩散（60s后四角毒雾每15s扩大1格）
  - 太空站 — 重力反转（每15s上下翻转）
  - 密林迷宫 — 荆棘陷阱（每20s随机激活3处，碰触扣血）

### 修改
- Player 新增 power-up 状态（power_timer / speed_timer / damage_multiplier）
- GameMap 新增 tile 破坏/查询/碰撞重建/移动墙驱动
- BattleScene 集成新系统：拾取/环境/破坏/碎片渲染

### 技术细节

## v1.0.0 (2026-05-10)

### 新增
- 完整双人对战射击游戏框架，基于 Python + Pygame
- **6种武器**：手枪、冲锋枪、霰弹枪、突击步枪、狙击枪、火箭筒
- **5种道具**：手雷、闪光弹、烟雾弹、血包、护盾
- **5张地图**：训练靶场、城市废墟、地下堡垒、太空站、密林迷宫
- **2种游戏模式**：回合淘汰(BO7)、死斗(15击杀获胜)
- **完整游戏流程**：主菜单 → 地图/模式选择 → 对战 → 购买阶段 → 结算
- **分屏镜头系统**：玩家距离近时自动合并为单画面，远时水平分割
- **经济系统**：起始资金$3000，击杀+$1500，获胜+$2500，连败补贴
- **AI Bot**：简单/普通/困难三级难度，决策树 AI
- **移动机制**：冲刺（2倍速0.5秒，冷却3秒）、翻滚（无敌帧，冷却5秒）
- **物理弹道**：子弹飞行速度 + 墙壁反弹 + 火箭弹弧线下坠
- **粒子特效**：枪口闪光、爆炸火球、命中火花、死亡特效、烟雾扩散
- **屏幕震动**：狙击枪、火箭筒、爆炸触发
- **程序化音效**：每种武器独立枪声、爆炸声、命中音效，无需外部音频文件
- **漫画风格渲染**：粗描边角色、高光内圈、阴影投影
- **HUD系统**：血条、弹药、冷却条、击杀播报、回合比分
- **购买界面**：30秒限时，武器+道具选购，双玩家各自独立购买
- **存档系统**：JSON多存档，经验/等级/解锁
- **局外成长**：击杀+100XP，获胜+200XP，等级解锁皮肤/角色/地图
- **键位布局**：P1(WASD+E/Q/Shift/Space)，P2(方向键+/ ./RShift/RCtrl)
- **人机对战模式**：单人 vs AI，支持三种难度

### 技术细节
- Python 3.10+ / Pygame 2.x
- 纯程序化渲染，无外部图片资源依赖
- 程序化音效合成（方波/噪声）
- 场景状态机架构
- JSON 数据驱动（武器、地图配置）
